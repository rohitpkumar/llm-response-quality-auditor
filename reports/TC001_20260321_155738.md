## Prompt

> What is the difference between supervised and unsupervised machine learning?

## Response Audited

> Supervised learning trains a model on labelled data, where each input has a known output. The model learns to map inputs to outputs and can predict labels for new data. Examples include classification and regression. Unsupervised learning, on the other hand, works with unlabelled data and tries to find hidden patterns or structure. Common techniques include clustering (e.g. k-means) and dimensionality reduction (e.g. PCA). The key difference is that supervised learning requires labelled training data while unsupervised learning does not.
